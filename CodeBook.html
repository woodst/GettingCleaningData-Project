Summary

In creating the second tidy dataset for this assignment, ALL of the variable names have had parenthesis and commas stripped, if they had them in the first place. This means many of the columns have been renamed. The existing column for subjects has been renamed to “subject”, and the numeric activity coding has been replaced with descriptive activities and the column named “activity”. For each variable the mean has been calculated from the subset of each subject for each activity. The numeric data in each of the 561 numeric columns is one of these averages. The use of all 561 variables is predicated on the wording of our assignment, which asked for “the average of each variables for each activity for each subject”. There seemed to be a consensus developing in the forums that only the 66 variables from the first tidy data set needed to be used, but that seems like a leap, given the way part 5 was worded.

The 561 variables have been slightly renamed to avoid incompatibility with some of the functions and libraries. Specifically, each name was stripped of any parentheses and commas, and everything else was left. For this data set, there are 180 rows (30 subjects X 6 activities each) and 561 variables, each of which is a numeric data type that holds an average score summarized from a large dataset for each subject and each activity.

The larger data set from which this summary is derived is itself a consolidation of the training and test datasets that were included with the original Samsung data. These two datasets were merged as follows:
